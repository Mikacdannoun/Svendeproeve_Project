#Combat Analyzer Project

Monorepo structure

- Backend: Node + Express + Typescript
- Web: React + Typescript
- Mobile: Expo + React Native + Typescript
- Database: MySQL + Prisma

## Structure
apps/
    api/ -> backend
    web/ -> web client
    mobile -> expo app
packages/
    shared-types -> shared TS types
prisma/ -> Prisma schema + migrations
